Estructuras básicas de Programación

La estructura básica de programación se basa en tres estructuras de control: secuencia, selección e iteración. Estas estructuras se utilizan en la programación estructurada

Estructuras de control

  • Secuencia: Ejecución de instrucciones en orden lineal Selección: Decisiones basadas en condiciones (if/else) Iteración: Repetición de acciones mediante bucles (for/while)

    • Estructura de repeticion

  • Tienen un punto de partida inicial
  • Tienen una condición que se evalúa para ejecutar un número determinado de acciones
  • Permiten repetir una acción o grupo de acciones

    • Tipos de estructuras de repetición

  • Desde/para ("For"): Se utiliza cuando se conoce el número de elementos que se quiere recorrer
  • Mientras ("while"): Se repite mientras una condición sea verdadera
  • Hacer mientras ("do while"): Se repite hasta que la condición sea falsa

    • Variables de control

  • Contador: Se incrementa o decrementa en una cantidad fija para contabilizar el número de veces que se desea una operación
  • Acumulador: Almacena el resultado de una o varias operaciones matemáticas

    • Usos

  • Para repetir una operación matemática
  • Para escribir algo en pantalla un cierto número de veces
